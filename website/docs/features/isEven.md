# isEven

Check if a number is even or not.

## Usage

```tsx
import { isEven } from "lulinjs";

const result = isEven(35)
// Output: false
```

```jsx live
import { isEven } from "./lulin";

function IsEven() {
    return (
        <div>
            <h4>Result:</h4>
            <h2 style={{color: 'rgb(102, 212, 189)'}}>
                {isEven(35)}
            </h2>
        </div>
    );
}
```
